## Installation

1. Clone the repo.

```bash
git clone https://github.com/Avi98/azure-task-commenting-report.git
```

2.  install all the dependency.

```bash
cd pref-monitor-ci
npm install
```

3. run the project

```bash
npm run dev
```

## Build Extension

1. create build and pack

```bash
npm run packV1
```

or

```bash
npm run packV2
```
